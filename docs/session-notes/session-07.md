# Arrays –

This document defines the step-by-step structure for Arrays.

The goal is to help the learner:

- understand data storage
- work with collections of values
- build logic using arrays
- prepare for strings and collections

---

# Session Breakdown

## Session 1 – Introduction to Arrays

### Topics

- What is an array
- Why arrays are needed
- Array declaration
- Array initialization
- Index concept (0-based index)

### Examples

```java
int[] arr = new int[5];
int[] numbers = {10, 20, 30, 40, 50};
```

### Practice (Live)

- Print all elements
- Print first and last element
- Access element using index

---

## Session 2 – Traversing Arrays

### Topics

- Looping through arrays using for loop
- Using for-each loop

### Practice (Live)

- Print all elements
- Print elements in reverse
- Print only even elements

---

## Session 3 – Basic Operations

### Topics

- Sum of elements
- Average of elements
- Count elements

### Practice

- Sum of array
- Average
- Count even numbers

---

## Session 4 – Searching & Comparison

### Topics

- Linear search
- Finding element
- Comparing values

### Practice

- Check if element exists
- Find index of element
- Count occurrences

---

## Session 5 – Min/Max Problems

### Topics

- Find largest element
- Find smallest element

### Practice

- max value
- min value
- second largest

---

## Session 6 – Advanced Logic

### Topics

- Reverse array
- Copy array
- Remove duplicates (basic idea)

### Practice

- reverse array
- compare arrays
- merge arrays (basic)

---

# Outcome

After arrays, learner should be able to:

- store multiple values
- iterate efficiently
- apply logic on datasets
