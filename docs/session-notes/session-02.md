# Concepts Covered

## Why Loops Are Needed

Loops allow us to repeat operations without writing the same code multiple times.

Example problem:

Print numbers from 1 to 10.

Instead of writing 10 print statements, we use loops.

---

# For Loop

Example syntax:

```java
for(int i = 1; i <= 10; i++){
    System.out.println(i);
}
```

Explanation:

- initialization → `int i = 1`
- condition → `i <= 10`
- increment → `i++`

---

# While Loop

Example syntax:

```java
int i = 1;

while(i <= 10){
    System.out.println(i);
    i++;
}
```
