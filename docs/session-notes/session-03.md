# Concepts Covered

## Why Loops Are Needed

Loops allow us to execute a block of code multiple times without writing repeated statements.

Example problem:

Print numbers from **1 to 10**.

Without loops we would write:

```
System.out.println(1);
System.out.println(2);
System.out.println(3);
...
```

Loops automate this repetition.

---

# Types of Loops

## For Loop

Example syntax:

```java
for(int i = 1; i <= 10; i++){
    System.out.println(i);
}
```

Explanation:

- Initialization → `int i = 1`
- Condition → `i <= 10`
- Increment → `i++`

---

## While Loop

Example syntax:

```java
int i = 1;

while(i <= 10){
    System.out.println(i);
    i++;
}
```

The loop runs until the condition becomes false.
