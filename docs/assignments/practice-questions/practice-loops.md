# Practice Problems – Loops

This document contains practice problems to strengthen your understanding of:

- `for` loop
- `while` loop
- `for-each` loop
- Using loops together with conditional statements

The problems are arranged in **increasing order of difficulty**.

## 28. Armstrong Number Check

An **Armstrong number** is a number where the **sum of the cubes of its digits equals the number itself**.

Example

```
153
```

Explanation

```
1³ + 5³ + 3³ = 153
```

Example Output

```
Armstrong Number
```

## 30. Sum of Squares from 1 to N

Write a program that calculates the **sum of squares of numbers from 1 to N**.

Example Input

```
5
```

Example Output

```
55
```

Explanation

```
1² + 2² + 3² + 4² + 5²
= 1 + 4 + 9 + 16 + 25
= 55
```
