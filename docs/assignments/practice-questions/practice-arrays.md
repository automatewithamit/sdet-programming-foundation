# Array Assignments 

These problems are designed in increasing order of difficulty.  
Each problem introduces a new concept or logic.

---

# Level 1 – Basics

## 1. Print all elements

Input  
[10, 20, 30]

Output  
10 20 30

---

## 2. Print elements at even index

Input  
[10, 20, 30, 40]

Output  
10 30

Explanation  
Index starts from 0 → even index = 0, 2

---

## 3. Print elements in reverse order

Input  
[1, 2, 3]

Output  
3 2 1

---

## 4. Find sum of elements

Input  
[5, 10, 15]

Output  
30

---

## 5. Find maximum element

Input  
[10, 50, 30]

Output  
50

---
