# Week 1 Assignments

## Session 1 Assignments

### 1. Even or Odd

Write a program that checks whether a number is even or odd.

Example

Input: 8  
Output: Even

---

### 2. Largest of Two Numbers

Write a program that finds the largest of two numbers.

Example

Input: 10, 20  
Output: 20

---

### 3. User Input Program

Write a program that takes input from the user and prints the value.

Example

Input: Sonal  
Output: Hello Sonal
